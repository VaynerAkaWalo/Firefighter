Firefighter minigame for developers
Basic implementation of firefighter, extinguishes forest on random coordinates every 25ms
The Colored square in front of you represents the burning forest, as a developer you can use your skills to help extinguish it.

The board is made of 50x50 individual cells, so the coordinates are actually an index in two-dimensional array, therefore values other than 0-49 are ignored.
